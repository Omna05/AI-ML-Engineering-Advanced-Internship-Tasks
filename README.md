This repository contains the completed work for Tasks 1 to 5 of the DevelopersHub AI Internship program. 

Task 1: Exploratory Data Analysis (EDA)
Objective
Understand the dataset, identify patterns, distributions, correlations, and potential data quality issues.

Work Done
-Loaded and inspected the dataset
-Checked missing values and data types
-Visualized feature distributions and correlations
-Identified key features influencing the target variable

Outcome
Built a strong understanding of the dataset structure and relationships, forming the foundation for modeling.

Task 2: Machine Learning Model (Tabular Data)
Objective
Build a predictive model using only tabular data.

Work Done
-Data preprocessing (encoding + scaling)
-Train-test split
-Trained regression models
-Evaluated performance using:
  MAE
  RMSE

Outcome
Established a baseline model for prediction using traditional machine learning techniques.

Task 3: Multimodal Learning (Images + Tabular Data)
Objective
Predict housing prices using both images and tabular data.

Methodology
-Tabular feature preprocessing and normalization
-Image feature extraction using pretrained ResNet18 (CNN)
-Fusion of image embeddings with tabular features
-Regression using Random Forest

Evaluation Metrics
Mean Absolute Error (MAE)
Root Mean Squared Error (RMSE)

Outcome
Successfully implemented a multimodal learning pipeline, demonstrating how visual and numerical data together improve predictive modeling.

Task 4: Model Optimization & Evaluation
Objective
Improve model performance and evaluate robustness.

Work Done
-Hyperparameter tuning
-Model comparison
-Error analysis
-Performance visualization

Outcome
Improved prediction accuracy and gained insights into model strengths and weaknesses.

Task 5: Final Analysis & Insights
Objective
Summarize results, learning outcomes, and real-world relevance.

Work Done
-Consolidated results from all tasks
-Compared single-modal vs multimodal performance
-Discussed limitations and future improvements

Outcome
Demonstrated practical understanding of end-to-end AI workflows, from data analysis to advanced multimodal modeling.
